Build offline current predictions

The Currents plugin can show tidal current speed and direction for Canadian waters — but the live source needs an internet connection. This page builds a local copy so it keeps working offline, out where there's no signal.

When you press the button:

  1. It downloads several months of official current predictions for every Canadian current station.
  2. It fits a small prediction model for each one, right here on this server.
  3. From then on, the plugin predicts Canadian currents with no internet needed.

This takes 2–3 hours and only needs to be done once (re-run it now and then to refresh). You can close this page — the build keeps running. Progress shows below.

idle

Where the data comes from

The predictions are published by the Canadian Hydrographic Service (CHS) — the government agency responsible for Canada's official tide and current data. Building the model here means the plugin doesn't have to reach their servers every time you want a reading.

The models are built under your own personal, non-commercial CHS licence and stay on this server — please don't redistribute them, as they contain Crown-copyright data. Not for navigation.
